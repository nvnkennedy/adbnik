Windows installer notice — unsigned build

This installer is NOT digitally signed. Microsoft Defender SmartScreen may show a warning such as "Windows protected your PC" or "Unknown publisher."

That warning appears because the file does not have an Authenticode certificate yet—not because the package was detected as malware. Unsigned installers are common for small open-source projects.

How to run anyway (SmartScreen / blue dialog):

  1. On the SmartScreen dialog, click "More info" (sometimes "More details").
  2. Then click "Run anyway".

If your browser blocked the download: open Downloads, right-click the file, Properties, check "Unblock" if present, then OK — then run the installer again.

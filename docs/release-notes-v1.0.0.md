# Adbnik 1.0.0

> **Current release:** see **[release-notes-v6.1.3.md](release-notes-v6.1.3.md)** (**v6.1.3**).

**Release date:** 2026-05-03

First **public** release under **v1.0.0** versioning (Apache-2.0).

## Install

### pip (all platforms)

```bash
python -m pip install adbnik
```

PyPI: [pypi.org/project/adbnik](https://pypi.org/project/adbnik/)

### Windows installer (GitHub — unsigned)

Download **`Adbnik-1.0.0-Setup-unsigned.exe`** from [GitHub Releases](https://github.com/nvnkennedy/adbnik/releases/tag/v1.0.0). The file is **not code-signed**; **SmartScreen** may block it — choose **More info → Run anyway**. A portable zip is also attached to the same release.

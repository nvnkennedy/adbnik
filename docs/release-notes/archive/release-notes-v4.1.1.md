# Adbnik 4.1.1

**Release date:** 2026-05-03

## Website

The **marketing site** (`https://nvnkennedy.github.io/adbnik/`) now exposes **User guide** on **every** primary page (including **Install**): the link always opens the GitHub-rendered guide:

**https://github.com/nvnkennedy/adbnik/blob/main/docs/guide/index.html**

Previously, **Install** (`install.html`) did not include that nav entry, so visitors could not reach the guide from that page without returning home.

## Application

No Python/API changes from 4.0.1; version bump aligns PyPI and Help → About with the site fix.

## Install

```bash
py -m pip install --upgrade adbnik
```

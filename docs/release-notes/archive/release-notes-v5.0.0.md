# Adbnik 5.0.0

**Release date:** 2026-05-03

## Terminal interaction

- **CMD** and **PowerShell** in the sidebar require a **double-click** to start Command Prompt or Windows PowerShell (single click is ignored).
- **Saved session bookmarks** (Terminal sidebar and **Login…** list) no longer **activate / open on a single click** on Windows; use **double-click** to open, or **Load into form** / context menu as before.

## Install

```bash
py -m pip install --upgrade adbnik
```

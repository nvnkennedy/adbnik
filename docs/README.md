# GitHub Pages (this folder)

The project site lives here: `index.html` and `css/`.

**Enable Pages:** Repository **Settings → Pages → Build and deployment → Source:** Branch **main**, folder **`/docs`**, then save. The site URL is **https://nvnkennedy.github.io/adbnik/** (after a short build).

`/.nojekyll` disables Jekyll so static files are served as-is.

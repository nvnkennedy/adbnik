Android Debugging Workspace

DeviceDeck is a desktop tool for ADB shell sessions, file operations, and screen control with scrcpy.

Production-ready packaging with Windows installer and portable builds.

Get DeviceDeck Read Docs

Highlights

  • Terminal sessions: ADB, SSH, serial, and local shell.
  • WinSCP-style local/remote file explorer workflows.
  • Screen mirror controls with scrcpy integration.

For enterprise or long-term usage, pin versions via GitHub Releases.

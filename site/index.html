One workspace for ADB, SSH, serial, files, and screen

DeviceDeck is a MobaXterm-style desktop workspace for Windows: a tabbed terminal for ADB shell, SSH (OpenSSH), and serial/COM sessions, plus local CMD/PowerShell — alongside an Android file explorer and scrcpy-based screen control.

Features

ADB & Android shell

Pick a device, open an interactive ADB shell tab, and use bookmarks and shortcuts for common tasks (root, remount, reboot, and more).

SSH terminal

Open SSH sessions from scratch or reuse SFTP fields from the file explorer. Uses the system OpenSSH client (ssh on your PATH).

Serial / COM console

Attach to COM ports at your chosen baud rate for boards and firmware logs — serial tabs alongside ADB and SSH in the same workspace.

File explorer

Browse Android device storage and transfer files; SFTP-oriented fields pair with SSH workflows when you work with remote hosts.

Screen mirroring

Launch scrcpy from the app when you need to see and control the device display on your PC.

Android features need ADB and scrcpy on your PATH. SSH needs the OpenSSH client (ssh). Serial uses COM hardware; the packaged app includes serial terminal support — set port and baud in the app or preferences.

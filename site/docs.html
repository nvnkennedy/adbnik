Install

  1. Install `DeviceDeck_Setup_*.exe`.
  2. Open DeviceDeck.
  3. Set ADB/scrcpy paths in Preferences if needed.

Quick start

  • Select Android device in top bar.
  • Open terminal session (ADB/SSH/PowerShell).
  • Use File Explorer tab for push/pull operations.
  • Use Screen Control tab to start scrcpy.

Troubleshooting

  • ADB not found: update path in Preferences.
  • No device: run `adb devices` and authorize USB debugging.
  • scrcpy fails: verify scrcpy binary and cable/device state.

For persistent issues, open a GitHub bug report with logs and reproduction steps.

# Downloads folder

DeviceDeck is **not** distributed as checked-in `.exe` / `.zip` files in this repository.

- **Users:** install with **`pip install devicedeck`** from [PyPI](https://pypi.org/project/devicedeck/). See the site **`install.html`**.
- **Maintainers:** optional PyInstaller builds stay **outside** git (see root `README.md`). This folder is intentionally empty in source control.

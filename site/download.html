Before you install

  • Windows 10 or later (64-bit).
  • Android / ADB / screen: install Google Android SDK platform-tools (for ADB) and a USB display-forwarding tool for live device screen access; add both folders to your PATH. Enable USB debugging on phones and tablets you attach.
  • SSH terminal: ensure an ssh client is installed and on your PATH (for example, the optional SSH client feature in Windows). DeviceDeck starts that client for SSH tabs.
  • Serial terminal: connect your COM device; choose port and baud in DeviceDeck (the packaged app includes serial support — no extra Python packages for end users).

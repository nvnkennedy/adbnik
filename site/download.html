Before you install

  • Windows 10 or later (64-bit).
  • Android / ADB / screen: install Android SDK Platform Tools (ADB) and scrcpy; add both to your PATH. Enable USB debugging on phones/tablets you attach.
  • SSH terminal: install Windows OpenSSH Client or ensure ssh.exe is on your PATH (DeviceDeck invokes it for SSH tabs).
  • Serial terminal: connect your COM device; choose port and baud in DeviceDeck (no extra install for serial in the packaged build).

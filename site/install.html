Install from PyPI

PyPI package: adbnik

Adbnik is published on PyPI as adbnik. Install with pip; the adbnik command is added to your environment (including Scripts on Windows).

Quick install

Requires Python 3.9+. Use a virtual environment if you like.

python -m pip install --upgrade pip
python -m pip install adbnik

Then run:

adbnik

Equivalent: python -m adbnik.

PyPI project page

Version history and metadata:

pypi.org/project/adbnik

Before you run

  • Android / ADB / screen: install platform-tools (ADB) and your USB display-forwarding utility; add to PATH, or set paths in File → Preferences.
  • SSH: an ssh client on PATH for SSH tabs.
  • Serial: COM hardware; set port and baud in the app.
